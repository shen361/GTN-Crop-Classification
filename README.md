The `RGB to_csv_batch_bands.py` script generates a time-series file of the study area. Then, `label_process.py` generates corresponding CSV files with labels for the labeled shapefiles. `create_dataset_based_on_label_2.py` generates a training set, which is then used in the training code to generate model weights. The prediction code inputs the time-series file of the prediction area to obtain the output CSV file. Finally, `output2image` generates a TIFF image, which is then projected onto the target area.
